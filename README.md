# FlickrClient
A simple flickr client 

After you download the project please add your flickr API_KEY in local.properties

```
API_KEY = "YOUR_API_KEY"
```


##Dependencies
1. Retrofit
2. Gson
3. Picasso
4. ButterKnife
5. Realm
6.TouchImageView
